# Лабораторная работа №4

## Описание

Консольное приложение на языке C#, разработанное в рамках дисциплины по объектно-ориентированному программированию.

Проект демонстрирует применение принципов ООП при разработке более сложной предметной модели с использованием собственных классов, интерфейсов и коллекций.

## Цель работы

- закрепление принципов объектно-ориентированного программирования;
- проектирование собственной предметной области;
- работа с коллекциями объектов;
- организация взаимодействия между сущностями системы.

## Основные возможности

- создание объектов предметной области;
- обработка пользовательских данных;
- выполнение операций над объектами;
- вывод информации в консоль;
- демонстрация взаимодействия между классами.

## Используемые технологии

- C#
- .NET
- Console Application
- Collections
- Object-Oriented Programming

## Использованные принципы ООП

- Инкапсуляция
- Наследование
- Полиморфизм
- Абстракция
- Интерфейсы

## Запуск проекта

### Visual Studio

Открыть решение и нажать **Start**.

### Через .NET CLI

```bash
dotnet restore
dotnet run
```

## Особенности проекта

Проект реализован с соблюдением принципов объектно-ориентированного программирования и демонстрирует разделение логики между различными классами.

Использование отдельных сущностей облегчает расширение функциональности и повторное использование кода.

## Возможные улучшения

- разделить проект на несколько файлов и пространств имён;
- добавить обработку пользовательского ввода;
- реализовать сохранение данных;
- добавить модульные тесты;
- использовать LINQ для обработки коллекций.
